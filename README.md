# 📊 Loan Analysis Dashboard – Power BI

## 🚀 Overview

This project presents a **business-focused Power BI dashboard** built to analyze loan performance and generate actionable insights.
It leverages **MTD (Month-to-Date)** and **MoM (Month-over-Month)** metrics to track trends, monitor KPIs, and identify risk patterns in lending data.

---

## 🎯 Objective

To transform raw financial loan data into a **decision-support dashboard** that helps:

* Track monthly performance
* Identify high-risk loans
* Monitor key financial KPIs

---

## 📌 Key Features

### 📅 Time Intelligence Analysis

* MTD (Month-to-Date) performance tracking
* MoM (Month-over-Month) growth comparison

### 📊 KPI Monitoring

* Total Loan Applications
* Funded Amount & Received Amount
* Average Interest Rate
* Loan Status Distribution

### ⚖️ Risk Segmentation

* Classification of **Good Loans vs Bad Loans**
* Identification of high-risk segments

### 🎛️ Interactive Dashboard

* Dynamic filters and slicers
* Drill-down capabilities
* Clean and user-friendly design

---

## 🧠 Key Insights

* Clear visibility into **monthly performance trends**
* Identification of **growth and decline patterns**
* Better understanding of **loan risk distribution**
* Data-driven support for financial decision-making

---

## 🛠️ Tools & Technologies

* **Power BI** – Dashboard development & visualization
* **DAX** – Time intelligence (MTD, MoM) & KPI measures
* **Power Query** – Data cleaning & transformation
* **Data Modeling** – Relationships and schema design
